Инструкция по заполнению формы

1. Нажмите на любую ячейку для ее активации (подсветится синим).

2. Используйте клавиатуру для ввода символов в активную ячейку.

3. Для перехода к следующей ячейке используйте клавишу Tab или стрелку вправо.

4. Для перехода к предыдущей ячейке используйте Shift+Tab или стрелку влево.

5. Для удаления символа и перехода к предыдущей ячейке используйте Backspace.

6. Для выбора вариантов в полях "Пол", "Цель въезда" и других - просто нажмите на соответствующий квадрат.

7. Для очистки всей формы нажмите кнопку "Очистить форму".

Управление с клавиатуры:
• Tab / → - следующая ячейка
• Shift+Tab / ← - предыдущая ячейка
• Backspace - удалить символ и перейти к предыдущей ячейке
• Delete - удалить символ в текущей ячейке
• Пробел - ввести пробел
• Enter - перейти к следующей строке
